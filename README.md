# Sugar Cosmetics

A clone of the popular beauty e-commerce website, Sugar Cosmetics. Made using HTML, CSS, JS, React and Redux. Includes features such as adding products to cart, applying discounts, category page listings etc.

If you would like to play around this project, follow the steps below:

1) Create a clone of the repository using the command

```
git clone https://github.com/devgurbir/sugarCosmetics
```

2) Install all the required packages

```
npm install
```

3) Run the frontend  on localhost

```
npm start
```
